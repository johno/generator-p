# generator-p

[![Build Status](https://secure.travis-ci.org/johnotander/generator-p.png?branch=master)](https://travis-ci.org/johnotander/generator-p)

A minimal generator for npm packages.

## Installation

```bash
npm install -g generator-p
```

## Usage

```bash
yo p
```
